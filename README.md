## Supported Representations

- UTF-8
- Bytes Array
- Hexa and Octal
- C Array
- SHA256 digest
- Integer
- INT ABS
- LOG10
- Scientific notation
- IPv4
- RGB Hexa + Decimal

## How to buid

```bash
cargo build
```

## How to run (interactive way)

```bash
cargo run
```
